# TradingAgents 首页商业化升级方案（ClaudeCode 执行版）

# 一、项目目标

当前首页已经具备：

* AI 投资终端视觉
* 多 Agent 信息结构
* 风险系统
* AI 机会雷达
* 深色专业风格

本次改造：

不是重做页面。

而是：

# 增加商业化结构。

最终目标：

将首页升级为：

```txt
Public Research Terminal
+
Private AI Workspace
```

网站形成：

# 公版内容

*

# 私版 AI 能力

的明确产品层级。

---

# 二、整体原则（必须遵守）

必须：

保留当前设计语言。

禁止：

* 推翻布局
* 改变深色体系
* 使用花哨科技风
* 使用高饱和彩虹渐变
* 使用卡通 AI 风格

整体风格：

必须保持：

```txt
机构级 AI 投资终端
AI Hedge Fund Workspace
Bloomberg Terminal
OpenAI Operator
```

方向。
