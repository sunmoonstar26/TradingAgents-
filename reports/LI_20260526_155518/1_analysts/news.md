## 一、地缘政治局势：中东危机主导全球风险情绪

过去一周，全球市场最核心的变量是**美国-伊朗冲突的持续升级**。尽管此前市场对伊朗和平协议抱有期待，但最新动态显示局势远未缓和：

| 关键事件 | 影响 |
|---------|------|
| 美军对伊朗南部发动"自卫打击"，攻击伊朗船只和导弹发射场 | 油价飙升，风险资产承压 |
| 伊朗阿巴斯港传出爆炸声 | 霍尔木兹海峡通航风险加剧 |
| 美国国务卿鲁比奥表示霍尔木兹海峡"必须以某种方式保持开放" | 暗示可能采取军事行动确保通航 |
| 伊朗表示和平协议签署后30天才开放霍尔木兹海峡 | 短期供应中断风险持续 |
| 卡塔尔能源公司延长不可抗力至8月中旬 | 天然气供应持续紧张 |

**对LI的影响路径**：中东紧张局势 → 油价上涨 → 中国汽油价格承压 → 消费者购车意愿可能受抑，但另一方面，高油价反而可能加速新能源车替代需求。

## 三、理想汽车（LI）专项分析

### 3.1 板块动态
| 事件 | 解读 |
|------|------|
| **蔚来（NIO）财报后股价大跌** | NIO业绩不及预期，引发对中国新能源车板块的整体悲观情绪。市场目光转向小鹏（XPeng）和理想汽车（LI） |
| **上周大盘下跌中EV板块领跌** | 大型股跌幅居前，EV板块受制于弱指引、监管担忧、债券收益率上升和竞争加剧 |
| **理想汽车被认为"仍被低估"** | 摩根士丹利给予理想汽车61%的上涨空间，认为当前价值评分上升但股价未能反映 |

### 3.2 关键数据要点
- **摩根士丹利目标价**：隐含61%上行空间 → 当前估值处于历史低位区间
- **竞争格局**：蔚来疲弱 → 市场份额可能向理想汽车和小鹏转移
- **Z评分上升**：价值指标改善，但市场尚未充分定价

### 3.3 风险提示
- **油价飙升**：虽长期利好新能源车，但短期可能压制消费者信心和购车预算
- **中美关系**：台海紧张局势升温，可能影响投资者对中国ADR的信心
- **整体市场避险情绪**：地缘不确定性可能导致资金从成长股（包括EV）撤出
